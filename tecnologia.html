A Trabin Software & Consulting tem como principal objetivo proporcionar a seus clientes os melhores produtos e serviços que podem ser obtidos através das mais recentes, atualizadas e consagradas tecnologias. Para conseguir isto selecionamos diversas tecnologias extremamente bem conceituadas no mercado mundial.
Os desenvolvedores Delphi sabem o valor de equipes pequenas e ágeis. Por anos eles se destacam quando comparados a empresas de maior porte, seja habilidade, expertise, ou na escolha correta de ferramentas. O Delphi oferece o "potencializador de força" definitivo para a produtividade, eliminando a morosidade e maximizando a eficiência do código.
O PHP (um acrônimo recursivo para "PHP: Hypertext Preprocessor") é uma linguagem de script Open Source de uso geral, muito utilizada e especialmente guarnecida para o desenvolvimento de aplicações Web embutivel dentro do HTML.

Utilizando a linguagem PHP você permite a interação direta do usuário com o site, visto que as informações são processadas em tempo real, juntamente com as de outros usuários, retornando em páginas HTML dinãmicas geradas para cada usuário. As aplicações em PHP são geradas automaticamente pelo servidor e com excelente performance, além de suportar a maioria das plataformas provedoras de acesso à Internet

No que se refere à segurança, é importante lembrar que o código PHP não é visto pelo usuário.
Sabemos que hoje em dia é cada vez mais importante ter um sistema flexível principalmente tratando-se de relatórios.

O ReportBuilder é uma das mais completas ferramentas para geração de relatórios do mercado para desenvolvedores Delphi. Criado pela empresa Digital Metaphors Corporation, o ReportBuilder é 100% escrito em Delphi, possuindo suporte para todas as suas versões.
O PostgreSQL é um poderoso sistema de banco de dados relacional open source. Ele tem mais de 15 anos de desenvolvimento ativo e oferece uma arquitetura com forte reputação quanto a confiabilidade, integridade de dados e corretude. Ele possui distribuições para a grande maioria de sistemas operacionais, incluindo Linux, Unix e Windows.

Como um banco de dados de classe empresarial, o PostgreSQL possui características sofisticadas como controle de concorrência multi-versão (MVCC), recuperação point in time, transações aninhadas (savepoints), backups online, um sofisticados planejador/otimizador de consultas e write ahead logging para tolerãncia a falhas. O PostgreSQL é altamente escal´vel tanto em sua capacidade de manipular grandes quantidades de dados como no gerenciamento do número de usuários simultâneos.
